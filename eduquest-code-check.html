EduQuestCode Check

Order Restored: verifying who actually finished each case.

Each student enters their name before starting. When they complete a case, the activity gives them a completion code built from that name. Paste your class list below and this tool shows you what every student's code should be for each of the four cases. A code that does not match is not a code. Nothing is stored anywhere: this runs entirely in your browser.

Paste a class list and press Generate.
Names are matched loosely: capitals, spaces, hyphens and apostrophes are all ignored, so hannah ferrier, Hannah Ferrier and HANNAH FERRIER all give the same code. But a nickname will not. If a student's code does not match, ask what they typed before you accuse them of anything.
